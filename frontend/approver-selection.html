Approver Selection

Assign an authorized approver to review and act on the document.

Select Approver

Every create and update action requires an approver selected from system users.

The creator must never be allowed to select themselves as approver.

Approval Rules

  • Only active users with approval rights should be listed.
  • Delegation rules should be logged if proxy approval is allowed.
  • Approver changes before submission must be tracked in audit trail.
